# Telephonic Interview Masterclass: 100 Questions for Full Stack Developers 📞

Welcome to the **Telephonic Screening Prep**. As a senior developer, when I take a phone interview, I look for **clarity**, **context**, and **foundational strength**. I don't just want the definition; I want to know if you've actually used these technologies in real-world scenarios.

This guide is split into 5 modules (20 questions each) to help you master the screening calls.

### 💡 Senior Developer Tips for Phone Calls:
1. **Be Concise**: Interviews are often timed (15-30 mins). Don't ramble.
2. **Context First**: Instead of saying "Redux is a store", say "We use Redux to manage global state across several complex components."
3. **Listen Carefully**: If an interviewer asks "Why", they want to hear about tradeoffs (Pros/Cons).
